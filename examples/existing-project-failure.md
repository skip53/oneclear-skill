# Example｜旧项目故障

## 业务员原始输入

客户说袋子用了半年压差就很高，清灰也没啥用，问是不是袋子质量不行。

## 客户原话

“这批袋子半年就不行了，压差一直高，喷吹也下不来，是不是你们袋子有问题？”

## Agent 分流判断

旧项目故障。当前 Level 2：可初步诊断，但不可选型。

## 工况档案草稿

- [已知] 故障表现是压差高，清灰后不下降。
- [已知] 使用时长约半年。
- [真实需求判断] 表层问题是“是不是袋子质量不行”，真实问题是压差高原因未定位。
- [异常状态] 存在责任归因压力，不能承认或否认质量问题。
- [风险升级等级] amber：有故障信号但缺证据链。
- [合规/承诺边界] 若进入索赔或质量责任判断，需要样品、照片、运行记录和人工确认。
- [商业/采购阶段] 故障索赔 / 责任判断阶段。
- [案件状态] diagnosis。
- [未知] 当前滤料材质、后处理、工况温度、含湿、粉尘粘性、清灰参数。
- [未知] 压差曲线、袋面照片、是否结露、是否工况近期变化。
- [风险] 存在糊袋、清灰异常、结露、粉尘粘性或过滤风速偏高风险。
- [限制] 不能直接判断为滤袋质量问题。

## 追问清单

### 内部技术版

1. 要求客户提供压差曲线或至少压差变化过程。
   - 优先级：P0
   - 为什么问：压差变化能区分逐步糊袋、突发堵塞或清灰系统异常。

2. 确认当前滤料材质、后处理、清灰方式、清灰压力和清灰周期。
   - 优先级：P1
   - 为什么问：这些信息影响糊袋和清灰效果判断。

3. 要求提供袋面照片或拆袋样品照片。
   - 优先级：P1
   - 为什么问：旧项目故障需要证据定位，单靠“压差高”不能判断材质问题。

### 客户转述版

1. 能不能发一下这半年压差大概怎么变化的？有没有记录或曲线？
2. 现场现在用的袋子是什么材质、什么后处理？喷吹压力和周期大概是多少？
3. 方便拍几张袋子表面的照片吗？最好有拆下来后的袋面状态。

## 初步诊断摘要

当前更像压差高 / 疑似糊袋或清灰相关故障。不能直接判断是滤袋质量问题。需要压差曲线、袋面照片、滤料信息、清灰参数和含湿/结露信息来继续诊断。

异常 / 合规 / 商业阶段：客户已提出质量责任问题，当前只能做证据需求和初步假设，不能进入责任结论或赔偿表述。

## 禁止承诺项

- 不承诺换材质一定解决。
- 不承认或否认质量问题。
- 不直接推荐替换方案。
