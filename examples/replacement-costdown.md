# Example｜替换 / 降本

## 业务员原始输入

客户现在用竞品 PTFE 覆膜袋，觉得太贵，问我们有没有便宜一点的，寿命差不多就行。

## 客户原话

“现在这个袋子能用，但是价格太高。你们有没有便宜点的，寿命差不多就行。”

## Agent 分流判断

替换 / 降本。当前 Level 2：可初步诊断，但不可选型。

## 工况档案草稿

- [已知] 当前方案能用，客户主要诉求是降本。
- [已知] 当前使用竞品 PTFE 覆膜袋。
- [真实需求判断] 表层需求是便宜替代，真实商业目标可能是降低总使用成本，而不是单价最低。
- [异常状态] 当前稳定方案要求降本，若无试用边界不能直接替换。
- [风险升级等级] amber：替换基线和试用边界缺失。
- [合规/承诺边界] 不能承诺同等寿命或完全替代竞品。
- [商业/采购阶段] 竞品替换 / 降本阶段。
- [案件状态] diagnosis。
- [未知] 当前寿命、价格区间、行业工艺、温度、含湿、粉尘性质、排放要求。
- [未知] 客户是否接受小批量试用。
- [风险] 存在低价替代后寿命或排放不稳定风险。

## 售前推进判断

- [决策链状态] 缺口：未确认现场使用者、技术评估人、采购负责人和最终决策人。
- [报价 readiness] 不能正式报价：缺规格、数量、工况边界和当前基线；只能列报价前置资料。
- [滤料能力匹配] 当前只能识别“替换降本风险”，不能形成候选方向。
- [试用/验证边界] 需要小批量试用；必须先确认当前稳定基线、试用范围、观察周期和验收指标。
- [竞品应对] 需当前基线 + 小批量试用，不用低价直接打竞品。
- [商业价值量化] 从单价转为总使用成本：寿命、停机、换袋人工、压差能耗、排放风险都未量化。
- [内部升级包] 暂不进入 Level 4；若客户要求同寿命或正式替换承诺，需要技术负责人确认。
- [禁止承诺项] 不承诺低价同寿命，不承诺完全替代竞品，不建议未试用直接全量替换。
- [下一步行动清单] 销售负责人在下一轮沟通前补当前寿命、价格区间、压差/排放基线、决策链和是否接受小批量试用；目标 Level：Level 3。

## 追问清单

### 内部技术版

1. 确认当前竞品方案实际寿命、价格区间和排放表现。
   - 优先级：P0
   - 为什么问：不知道当前稳定基线，就不能判断替换是否划算。

2. 确认客户降本目标是单价下降、总成本下降，还是交期/服务改善。
   - 优先级：P1
   - 为什么问：单价降本和总成本降本对应的替代策略不同。

3. 确认客户是否接受分仓或小批量试用。
   - 优先级：P1
   - 为什么问：稳定方案降本不应直接全量替换，需要控制试用风险。

### 客户转述版

1. 现在这款袋子大概能用多久？排放和压差表现稳定吗？
2. 客户想降的是采购单价，还是希望整体使用成本更低？
3. 如果我们做替代，客户能不能先接受小批量或分仓试用？

## 初步诊断摘要

这是稳定方案降本诉求，不是故障优先。当前不能承诺便宜方案同等寿命。下一步要确认当前基线和客户是否接受试用。

需求 gap：客户说“便宜点、寿命差不多”，但还没有当前寿命、价格、排放和试用边界，不能直接转成替代承诺。

## 禁止承诺项

- 不承诺低价同寿命。
- 不承诺完全替代竞品。
- 不建议未试用直接全量替换。
