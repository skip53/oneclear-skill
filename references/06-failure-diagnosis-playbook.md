# 06｜旧项目故障诊断 Playbook

旧项目故障不能直接归因于滤料材质。先看表现、位置、比例、运行变化和证据。
本文件每条故障都按「机制 + 对策 + 典型对话」三段式组织——读完一条，就知道**为什么不能直接下结论**、**要凑齐哪些证据才能开口**、**应该怎么回应业务员**。

> 通用自检（任何故障都用得上）：开口给方向前必须确认 ① 客户原话与业务员转述是否一致 ② 是否已有照片/曲线/检测报告 ③ 现场是否承认存在多种可能原因。三条任一缺失，先反问证据，不归因。

## 糊袋

- 典型表现：压差升高、清灰后不下降、袋面粉尘粘附。
- 优先追问：含湿、结露、粉尘粘性/油性、清灰压力、启停频率、糊袋照片。
- 可能原因：结露、粉尘粘性、清灰不足、过滤风速偏高、后处理不匹配。
- 需要证据：压差曲线、袋面照片、温度记录、粉尘样品。
  - **如何拿到**：
    - 压差曲线 → DCS/PLC 历史导出（连续 ≥ 7 天，标注清灰时刻）
    - 袋面照片 → 现场手机拍摄（糊袋区域特写 + 整袋全景 + 袋口/袋身/袋底各 1 张）
    - 温度记录 → DCS/PLC 历史导出 OR 现场巡检记录
    - 粉尘样品 → 客户处取 ≥ 50g 装入密封袋（标注取样位置 + 日期）

- **机制（为什么不能直接归因滤料）**：糊袋是「粉尘/含湿/清灰/风速/后处理」5 类因素叠加的**结果**，不是滤料本身失效的证据；其中至少 3 类（清灰不足、过滤风速高、含湿结露）与滤料无关。若只凭压差高 + 袋面粘附直接归因到「滤料不亲水/不拒油」，会让客户把系统改造该做的清灰压力/喷吹周期/含湿源头治理工作全部跳过，**只换滤料后糊袋再现**的概率 > 70%。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **压差曲线**（连续 ≥ 7 天、标注清灰时刻）→ 区分「清灰无效型糊袋」（清灰后压差不降）vs「运行中累积型糊袋」（清灰后短期下降、再缓慢爬升）。
  2. **清灰系统记录**（喷吹压力、周期、脉冲阀最近一次检修时间）→ 排除脉冲阀膜片破损/压力不足/管路积水等纯机械问题。
  3. **含湿量 + 入口温度 + 露点**（任一缺失即止步）→ 含湿 > 5% 且入口温度接近露点是糊袋的**充分条件**，与滤料无关。
- **典型对话**：
  - 业务员：「客户压差高、袋面粘附，是不是滤料拒油性不够？」
  - ✗ 错误回应：「建议换 PTFE 覆膜 + 拒油后处理。」
  - ✓ 正确回应：「糊袋有 5 类可能原因，其中至少 3 类与滤料无关。请先发 ① 过去 7 天压差曲线（标注清灰时刻） ② 当前清灰压力/周期/脉冲阀状态 ③ 入口温度 + 含湿量 + 露点。凑齐 2 项后我再给方向。」

## 破袋

- 典型表现：排放升高、局部破损、袋口/袋底/迎风面破损。
- 优先追问：破损位置、比例、粉尘磨蚀性、安装情况、骨架情况、过滤风速、照片。
- 可能原因：磨蚀、安装问题、骨架毛刺、气流冲刷、火星、高温、材质老化。
- 需要证据：破损照片、位置分布、骨架检查、运行记录。
  - **如何拿到**：
    - 破损照片 → 现场手机拍摄 ≥ 3 张（袋口/袋身迎风面/袋身背风面/袋底/缝合处，标注位置）
    - 位置分布 → 让客户清点同批破袋数量 + 每条破损位置（按上面 5 个区域分类）
    - 骨架检查 → 抽出破损袋对应的骨架，毛刺/变形/间距/与袋口贴合度拍照
    - 运行记录 → 同批滤袋启用日期 + 当前运行月数 + 启停频次

- **机制（为什么不能直接归因滤料）**：破袋是**失效位置可观察的物理损伤**，但损伤位置本身就锁定了根因方向——袋口破损多为骨架/安装问题、袋底破损多为气流分布/灰斗问题、迎风面破损多为磨蚀/风速问题。**滤料材质问题（老化、腐蚀、水解）通常呈现为大面积均匀变脆、强度普降，不会只破在某个固定位置**。若不先看破损位置就归因滤料，会让客户把应当检修的骨架/气流分布/花板问题全部跳过。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **破损位置分布**（袋口 / 袋身迎风面 / 袋身背风面 / 袋底 / 缝合处，附 ≥ 3 张实拍）→ 位置直接决定根因方向。
  2. **骨架检查**（毛刺、变形、间距、与袋口贴合度）→ 袋口破损 80% 与骨架有关，与滤料无关。
  3. **同批破袋比例 + 运行时长**（破袋比例 < 5% 偏安装/局部问题；> 30% 偏材质/工况普适问题）→ 决定是单点修复还是全量替换。
- **典型对话**：
  - 业务员：「客户破袋了，是不是滤料质量不行？」
  - ✗ 错误回应：「建议升级芳纶/PTFE 材质。」
  - ✓ 正确回应：「破袋位置在哪？袋口/袋身/袋底？同批破袋比例多少？请发 ≥ 3 张破损部位实拍 + 骨架检查照片 + 这批滤袋运行了多久。位置和比例决定根因是安装、骨架、磨蚀还是材质。」

## 压差高

- 典型表现：系统阻力上升、风量下降、清灰频繁。
- 优先追问：压差曲线、清灰参数、含湿、粉尘粘性、运行温度、滤袋使用时长。
- 可能原因：糊袋、清灰系统异常、粉尘负荷变化、过滤风速偏高、结露。
- 需要证据：压差曲线、清灰系统记录、袋面照片。
  - **如何拿到**：
    - 压差曲线 → DCS/PLC 历史导出（连续 ≥ 7 天，标注清灰时刻）
    - 清灰系统记录 → 喷吹压力表读数 + 喷吹周期设置 + 脉冲阀最近一次检修日期
    - 袋面照片 → 现场手机拍摄（糊袋区域特写 + 整袋全景）

- **机制（为什么不能直接归因滤料）**：压差是系统阻力的**综合读数**，由「粉尘层 + 清灰残留 + 含湿 + 风速 + 结露」多因叠加。其中至少 4 项（清灰残留/含湿/风速/结露）与滤料本身无关。若只凭压差数据归因滤料材质，会引导客户把工况系统问题（清灰/风速/含湿）误投到材质升级，**错失真正的修复点**。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **压差曲线**（连续 ≥ 7 天、标注清灰时刻）→ 区分「清灰无效」（清灰后压差不降）vs「清灰后压差仍上升」（累积型）。
  2. **清灰参数记录**（喷吹压力、周期、脉冲阀状态）→ 排除清灰系统问题。
  3. **含湿量 + 入口温度 + 露点**（任一缺失即止步）→ 排除结露型糊袋。
- **典型对话**：
  - 业务员：「客户压差高，是不是滤料不行？」
  - ✗ 错误回应：「建议升级覆膜滤料。」
  - ✓ 正确回应：「压差高可能由 5 种原因引起。请先发 ① 过去 7 天压差曲线（标注清灰时刻） ② 当前清灰压力/周期/脉冲阀状态 ③ 入口温度 + 含湿量 + 露点。凑齐 2 项后我再给方向。」

## 排放超标

- 典型表现：出口粉尘浓度高、烟囱冒灰、在线监测异常。
- 优先追问：是否破袋、是否新装、是否旁路泄漏、滤料是否覆膜、粉尘细度、排放数据。
- 可能原因：破袋、安装密封问题、滤料精度不足、设备漏风、清灰过强。
- 需要证据：排放曲线、检漏记录、滤袋安装检查。
  - **如何拿到**：
    - 排放曲线 → CEMS 在线监测数据导出（超标时段前后 24h，含时间戳）
    - 检漏记录 → 粉尘检漏仪扫描原始数据 + 报告
    - 滤袋安装检查 → 现场开盖拍照花板密封、滤袋口与花板贴合、箱体可见破损点

- **机制（为什么不能直接归因滤料）**：排放超标有 ≥ 5 个独立根因（破袋、密封泄漏、旁路阀漏、设备漏风、清灰过强把粉尘层吹穿），其中仅「滤料精度不足」一项与滤料相关。**只换高精度滤料、不修密封和破袋，排放数据可能纹丝不动**。更危险的是：若现场已有破袋（破袋后排放超标是必然结果），再换更高精度滤料会误导现场把"已破袋"误判为"滤料不够好"，延误更换破损袋的时机。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **排放连续曲线**（含超标时段前后 24 小时）→ 区分「突发超标」（多与破袋/旁路泄漏有关）vs「渐进超标」（多与滤料堵塞/粉尘层变化有关）。
  2. **破袋检查**（粉尘检漏仪扫描 + 花板/箱体可见破损排查）→ 破袋是单点修复问题，**不是换滤料能解决的**。
  3. **旁路阀 + 箱体密封检查**（旁路阀密封圈、箱体负压段漏风点）→ 漏风量 > 15% 时排放数据不可信，先修漏再谈滤料。
- **典型对话**：
  - 业务员：「客户排放超标，要求换更高精度滤料。」
  - ✗ 错误回应：「建议升级到 PTFE 覆膜 + 精度提升一级。」
  - ✓ 正确回应：「排放超标有 ≥ 5 个根因，其中 4 个与滤料无关。请先确认 ① 是否有破袋（检漏仪扫一遍） ② 旁路阀和箱体密封是否漏风 ③ 排放曲线是突发还是渐进。三个都查完，再决定要不要动滤料。」

## 寿命短

- 典型表现：未达到预期寿命就破损、糊袋、变脆或排放异常。
- 优先追问：目标寿命、实际寿命、材质、工况变化、失效形态、历史寿命。
- 可能原因：选型不匹配、工况变化、运行问题、清灰问题、腐蚀、水解、氧化、磨蚀。
- 需要证据：样品检测、破损形态、运行数据、历史对比。
  - **如何拿到**：
    - 样品检测 → 失效滤袋送检（红外/DSC/强度测试，附送检单）
    - 破损形态 → 现场手机拍摄（变脆/磨穿/破洞/糊死/腐蚀坑，附拍摄日期 + 倍数参照物）
    - 运行数据 → 工况逐月记录（温度/含湿/烟气成分/负荷）
    - 历史对比 → 同工况下这批滤袋的实际运行月数（不是销售合同写的预期寿命）

- **机制（为什么不能直接归因供应商）**：寿命短是**累计损伤的结果**，其根因分散在 4 个层面（选型匹配 / 工况变化 / 运行操作 / 材质性能），供应商只对「材质性能」这一层负责。若不先做失效形态分析（变脆 = 氧化/水解、磨穿 = 磨蚀/风速、破洞 = 火星/机械），就把寿命短归因到供应商质量，会引发两重损失：① 客户错过工况改善机会 ② 供应商承担了不属于自己的责任（一旦承诺换货，会被反向索赔）。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **失效形态分析**（变脆 / 磨穿 / 破洞 / 糊死 / 腐蚀坑，附照片）→ 形态直接对应根因方向。
  2. **工况变化记录**（温度、含湿、烟气成分、负荷的逐月变化，与初装时对比）→ 工况漂移是寿命短的头号原因。
  3. **同批滤袋历史寿命**（同工况下这批滤袋的真实运行月数，**不是销售合同写的预期寿命**）→ 区分「普遍寿命短」vs「这一批异常短」。
- **典型对话**：
  - 业务员：「客户用了 8 个月就破了，承诺是 18 个月，要求索赔。」
  - ✗ 错误回应：「建议按 18 个月寿命做质保方案。」
  - ✓ 正确回应：「索赔需要证据链。请先发 ① 失效形态照片（变脆/磨穿/破洞/糊死） ② 工况逐月记录（温度/含湿/烟气成分） ③ 这批滤袋同工况历史寿命。失效形态 + 工况漂移先定位，再谈质保。」

## 烧袋

- 典型表现：孔洞、烧穿、局部焦化。
- 优先追问：火星来源、峰值温度、高温颗粒、阻火措施、烧损位置。
- 可能原因：火星进入、温度峰值过高、预处理不足。
- 需要证据：烧损照片、温度记录、火星捕集记录。
  - **如何拿到**：
    - 烧损照片 → 现场手机拍摄（孔洞/烧穿/局部焦化，附拍摄日期 + 位置标注）
    - 温度记录 → DCS/PLC 历史导出（最近 30 天峰值温度 + 持续时长）
    - 火星捕集记录 → 上游火花捕集器/旋风/喷淋塔的运行记录 + 最近维护日期

- **机制（为什么不能承诺单靠滤料解决）**：烧袋的本质是**火星/高温颗粒进入袋室**，滤料能提供的保护是有限的（PTFE 长期使用温度 ~260℃，瞬时可耐 280℃；超过即碳化、烧穿）。若上游阻火/火星捕集/喷淋降温等措施不到位，**任何滤料都会被烧穿**——换滤料不解决火星源头问题，只会把"短期烧穿"变成"短期再烧穿"。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **火星来源定位**（上游是否有火花捕集器/旋风除尘/喷淋塔，捕集器是否定期清灰）→ 火星源头在袋室之外，滤料管不到。
  2. **峰值温度实测**（含最近 30 天最高值 + 持续时长）→ 滤料选型的硬边界是峰值温度，**瞬时超温即烧穿**。
  3. **阻火措施现状**（是否有火花捕集器、阻火网、喷淋塔；最近一次维护日期）→ 缺位或失效 = 烧袋高风险，与滤料无关。
- **典型对话**：
  - 业务员：「客户烧袋了，要求换耐高温滤料。」
  - ✗ 错误回应：「建议升级到 PPS 或 PTFE 滤料。」
  - ✓ 正确回应：「烧袋的根因是火星/超温进入袋室，滤料能挡的有限。请先确认 ① 上游是否有火花捕集器/喷淋塔 ② 最近 30 天峰值温度 + 持续时长 ③ 阻火措施最近维护日期。火星源头不解决，换什么滤料都会再烧。」

## 跨故障通用检查清单

> 任何故障开口前先过一遍这 4 条。任一不达标，按"证据不足"处理，反问而不归因。

1. **客户原话 vs 业务员转述** 是否一致？业务员常把"压差高 + 排放高"压缩成"滤料不行"，需要拆开。
2. **照片 / 曲线 / 检测报告** 是否齐全？无图无真相，单凭描述不归因。
3. **现场是否承认多种可能**？若客户/业务员已锁定"就是滤料问题"，且拒绝看其他可能 → 列入 framing trap，不接受预判。
4. **同批/同工况历史** 是否可对比？没有历史基线就没有"寿命短/压差高"的标准。
