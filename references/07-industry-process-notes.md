# 07｜行业 / 工艺提示

This file provides starter prompts by industry. It is not a complete technical manual.

## 水泥

- 常见工艺段：窑尾、窑头、煤磨、水泥磨、包装、破碎。
- 常见问题：高温波动、结露糊袋、粉尘磨蚀、压差高、排放要求。
- 优先追问：工艺段、温度峰值、启停频率、含湿、粉尘细度、清灰方式。

## 钢铁

- 常见工艺段：烧结、炼铁、炼钢、转炉、电炉、焦化。
- 常见问题：高温、火星、磨蚀、排放、腐蚀。
- 优先追问：火星/高温颗粒、烟气成分、温度波动、粉尘磨蚀性。

## 垃圾焚烧

- 常见问题：酸性气体、腐蚀、含湿、温度波动、二噁英相关排放要求。
- 优先追问：HCl/SOx/NOx、含湿、温度、喷药系统、排放指标。

## 沥青

- 常见问题：油性、粘性、糊袋、温度、清灰。
- 优先追问：油烟比例、粉尘粘性、温度、清灰参数、糊袋照片。

## 化工

- 常见问题：腐蚀、易燃易爆、粉尘性质复杂、工况波动。
- 优先追问：化学成分、酸碱性、含湿、易燃易爆风险、安全要求。

## 电厂 / 锅炉

- 常见问题：温度、酸露点、粉尘负荷、排放、启停。
- 优先追问：燃料类型、烟气成分、温度、含湿、排放要求。

## 有色冶炼

- 常见问题：高温、腐蚀、重金属粉尘、磨蚀。
- 优先追问：烟气成分、温度、粉尘磨蚀性、腐蚀性、排放要求。

## 木工 / 家具

- 常见问题：粉尘易燃、静电、排放、清灰。
- 优先追问：粉尘爆炸风险、含湿、颗粒大小、清灰方式、安全配置。
