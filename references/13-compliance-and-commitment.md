# 13｜合规与承诺边界

工业滤料售前的“安全合规”主要不是软件数据安全，而是现场资料、环保排放、安环、质保、合同和招投标承诺边界。

## 合规/承诺类型

| 类型 | 触发信号 | Agent 处理 |
| --- | --- | --- |
| 环保排放 | 保证排放、超低排放、mg/Nm3 指标 | 只能记录目标和验证条件，不能保证 |
| 质保寿命 | 保用几年、保证寿命、同等寿命 | 标注需证据和人工确认 |
| 安环风险 | 火星、烧袋、爆炸、着火、安全事故 | 不承诺单靠滤料解决，要求系统措施 |
| 客户资料保密 | 现场照片、检测报告、运行记录 | 标注资料来源和使用范围 |
| 竞品资料保密 | 竞品样品、竞品报告、价格信息 | 不外传，不直接做不当比较 |
| 招投标合同 | 技术条款、偏差表、盖章承诺、合同附件 | 人工确认后才能输出确定承诺 |
| 索赔责任 | 是否质量问题、赔偿、责任归属 | 不承认不否认，要求证据链 |

## 禁止话术

- “我们保证排放达标。”
- “这个材质肯定能用两年。”
- “可以完全替代竞品。”
- “肯定不是我们的质量问题。”
- “直接写进技术协议没问题。”

## 允许话术

```text
目前可以把排放/寿命作为目标记录下来，但是否能承诺，需要结合完整工况、样品/报告、试用结果和技术负责人确认。
```

```text
客户提供的照片、样品和竞品资料需要限定内部技术判断用途，不建议直接外传或作为对外承诺依据。
```
