# 21｜商业价值量化

商业价值要从单价转为总使用成本。没有总成本口径时，Agent 只能列缺口，不能承诺降本幅度。

## 总使用成本维度

| 维度 | 需要的数据 | 价值解释 |
| --- | --- | --- |
| 滤袋采购单价 | 当前单价、目标单价、数量 | 只反映采购成本，不等于总成本 |
| 实际寿命 | 当前寿命、失效时间、目标寿命 | 影响年化换袋成本 |
| 停机成本 | 换袋停机时间、产线损失 | 短寿命会放大停机损失 |
| 人工换袋成本 | 人工数量、工时、外协费用 | 影响维护预算 |
| 压差 / 能耗 | 压差曲线、风机能耗 | 压差升高可能增加运行成本 |
| 排放 / 环保风险 | 排放记录、处罚风险、停产风险 | 不能承诺排放，只能记录风险价值 |
| 异常维护 | 糊袋、破袋、清灰异常频次 | 影响现场稳定性和管理成本 |

## 禁止承诺

- 不承诺“必然降本”。
- 不承诺“单价低就是更划算”。
- 不用没有基线的数据做 ROI。
- 不把排放或寿命目标写成保证。
