# 08｜业务员常见模糊表达

This file teaches the Agent how to handle messy salesperson inputs.

## “客户说袋子不耐用”

- 可能对应：寿命短、破袋、糊袋、排放超标、成本抱怨。
- 不能理解为：材质差。
- 优先追问：具体是不耐用在哪里？用了多久？怎么坏？有没有照片？

## “客户说压差高”

- 可能对应：糊袋、清灰异常、过滤风速偏高、粉尘变化、结露。
- 不能理解为：滤料透气差。
- 优先追问：压差曲线、清灰参数、含湿、粉尘粘性、运行温度。

## “客户想换便宜点”

- 可能对应：稳定方案降本、竞品替换、预算压力、服务不满。
- 不能理解为：只要低价替代。
- 优先追问：当前方案是否稳定？目标是降本还是解决问题？能否小批量试用？

## “客户说排放不行”

- 可能对应：破袋、安装密封、滤料精度、设备漏风、在线监测异常。
- 不能理解为：必须上更高等级滤料。
- 优先追问：排放数据、是否破袋、是否新装、是否有检漏、当前滤料是否覆膜。

## “客户让直接报价”

- 可能对应：已知方案复购、业务员未提供工况、客户只想比价。
- 不能理解为：可以跳过工况。
- 优先追问：是否已有规格和材质？是否同工况复购？是否需要技术判断？
