# 04｜风险标签

Risk tags are preliminary. A tag is not a final diagnosis.
本文件每条风险都按「机制 + 对策 + 典型对话」三段式组织——读完一条，就知道**为什么这个风险会让承诺失败**、**至少要补哪些 P0/P1 才能开口**、**应该怎么回应业务员**。

> 通用自检：每条风险标签触发时，**不要立即给承诺或型号建议**。先按对策反问证据，证据不齐就把这条标为"待证"——而不是"已确认"。

## 温度风险

- 触发信号：常态温度接近材料边界；峰值温度未知；客户提到烧袋、变形、收缩。
- 需要补充：常态温度、峰值温度、峰值持续时间、是否有火星。
- 不能承诺：不能承诺当前材料或替代材料长期安全。
- 建议证据：温度曲线、现场运行记录、破损照片。
  - **如何拿到**：
    - 温度曲线 → DCS/PLC 历史导出（CSV 或截图，含时间戳）OR 现场巡检记录本照片
    - 现场运行记录 → 客户处值班日志 / 启停记录 / 故障记录
    - 破损照片 → 现场手机拍摄（袋口、袋身、袋底各 1 张，附拍摄日期）

- **机制（为什么这个承诺会失败）**：滤料的耐温边界有「长期使用温度」和「瞬时峰值温度」**两条硬线**——长期超温 = 寿命衰减（强度下降、变脆）、瞬时超温 = 立即失效（烧穿、收缩）。若只知常态温度不知峰值温度就给承诺，承诺大概率在**第一个温度波动季**被打脸。同时，**「接近边界」的承诺安全裕度 = 0**——客户工况漂移 5–10℃ 是常态（负载变化、季节变化、停产再开），看似"够用"的余量在第二年往往不够。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **常态温度**（月均值 + 最高值 + 最低值，至少 30 天数据）→ 决定滤料长期使用温度档位（PPS 190℃ / 芳纶 204℃ / PTFE 260℃）。
  2. **峰值温度**（最近 90 天最大值 + 持续时长 + 频次）→ 决定是否需要抗瞬时超温设计。
  3. **火星 / 高温颗粒存在与否** → 有火星 = 必须叠加火星捕集措施（见 火星烧袋风险）。
- **典型对话**：
  - 业务员：「客户温度 180℃，推 PPS 滤袋行不行？」
  - ✗ 错误回应：「PPS 长期 190℃ 够用，可以推。」
  - ✓ 正确回应：「PPS 长期 190℃ 余量只有 10℃。请先发 ① 最近 30 天温度曲线（最高/最低/均值） ② 最近 90 天峰值温度 + 持续时长 + 频次 ③ 是否有火星。凑齐 2 项后我再判断 PPS 是否真够用。」

## 结露 / 含湿风险

- 触发信号：启停频繁、低温运行、糊袋、粉尘粘、含湿未知。
- 需要补充：含湿量、露点、入口温度、启停频率、保温情况。
- 不能承诺：不能承诺更换滤料即可解决糊袋。
- 建议证据：压差曲线、糊袋照片、运行温度记录。
  - **如何拿到**：
    - 压差曲线 → DCS/PLC 历史导出（连续 ≥ 7 天，标注清灰时刻）
    - 糊袋照片 → 现场手机拍摄（糊袋区域特写 + 整袋全景）
    - 运行温度记录 → DCS/PLC 历史导出 + 现场巡检记录本照片

- **机制（为什么这个承诺会失败）**：结露 = 入口温度低于露点 + 含湿。**这是充分条件型风险**——一旦发生结露，糊袋是 100% 物理必然（液滴在滤料表面架桥、粉尘粘附），与滤料是否亲水、是否覆膜、是否拒油**完全无关**。把结露型糊袋归因到滤料 = 让客户反复换滤料但糊袋永远再现（结露源头没治理）。含湿量 > 5% 且入口温度接近露点是糊袋的**充分条件**。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **含湿量**（体积百分比或 g/Nm³）→ > 5% 即进入结露高风险区。
  2. **入口温度 + 露点**（同步测量）→ 入口温度 ≤ 露点 + 3℃ = 临界，必须有保温或升温措施。
  3. **启停频率**（24h 内启停次数 + 每次停机时长）→ 频繁启停 = 反复穿越露点 = 结露累积。
- **典型对话**：
  - 业务员：「客户糊袋，是不是要换拒油后处理？」
  - ✗ 错误回应：「建议 PTFE 覆膜 + 拒油后处理。」
  - ✓ 正确回应：「糊袋是结果，不是原因。请先确认 ① 含湿量（g/Nm³ 或 %） ② 入口温度与露点的差值 ③ 24h 启停频率。结露型糊袋换什么滤料都糊，必须先解决温度/含湿/保温。」

## 水解风险

- 触发信号：高湿、高温、滤袋变脆、寿命短、材质可能对水解敏感。
- 需要补充：温度、含湿量、材质、使用时长。
- 不能承诺：不能在缺水分和温度数据时判断材质失效原因。
- 建议证据：样品检测、断裂形态、运行记录。
  - **如何拿到**：
    - 样品检测 → 失效滤袋送检（红外/DSC/拉伸强度，附送检单 + 标注取样位置）
    - 断裂形态 → 现场手机拍摄（变脆但保留形态 vs 变色 vs 糊死，附拍摄日期 + 倍数参照物）
    - 运行记录 → 工况逐月记录（温度/含湿/烟气成分） + 同批滤袋运行时长

- **机制（为什么这个承诺会失败）**：水解是**特定材质 + 高温高湿**的化学反应——PPS 在高温（> 150℃）+ 高湿（含湿 > 5%）环境下会逐步水解断链、强度下降、变脆。**水解后的滤料没有"修复"路径，只能换**。但水解失效的形态（变脆、强度下降、保留强度低）与氧化失效形态相似，**误判会导致把氧化问题当成水解（反之亦然）**——材质升级方向反了，下一批还会再水解。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **温度 + 含湿**（同时存在，且持续时长）→ PPS 在 > 150℃ + > 5% 含湿下水解速度指数级上升。
  2. **失效形态**（变脆但保留形状 = 水解典型；变脆 + 变色 = 氧化典型；糊死 = 结露/粘性）→ 形态锁定水解/氧化方向。
  3. **当前材质 + 运行月数**（PPS 8 个月变脆 vs 24 个月变脆，性质不同）→ 决定是材质不耐水解还是工况漂移触发了水解。
- **典型对话**：
  - 业务员：「客户 PPS 用 10 个月就变脆了，是不是水解？」
  - ✗ 错误回应：「建议改用 PTFE 防水解。」
  - ✓ 正确回应：「水解需要「高温 + 高湿」同时存在。请先发 ① 实际温度曲线（最近 6 个月） ② 含湿量 ③ 失效滤袋的变脆形态照片（保留形状 vs 变色 vs 糊死）。形态 + 温度 + 含湿三项凑齐，才能定方向。」

## 氧化风险

- 触发信号：高含氧、高温、材料强度下降、变脆。
- 需要补充：含氧量、温度、材质、运行时长。
- 不能承诺：不能只凭寿命短判断为氧化。
- 建议证据：化验报告、样品检测。
  - **如何拿到**：
    - 化验报告 → 烟气成分化验（袋室前取样，标注含氧量百分比 + 取样日期）
    - 样品检测 → 失效滤袋送检（红外/DSC，重点看变脆 + 变色形态）

- **机制（为什么这个承诺会失败）**：氧化 = 高含氧 + 高温（> 150℃）的化学反应，主要发生在 PPS（高氧敏感）、P84/PI 等材质。**氧化与水解形态相似（都变脆）但根因不同**——氧化要降含氧或降温，水解要降含湿；搞反了，工况改善方向就反了。同样，**只凭"变脆"无法区分氧化与水解**——必须有含氧量数据。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **含氧量**（体积百分比，袋室前测量）→ > 15% 即进入 PPS 氧化高风险区。
  2. **温度**（尤其峰值 + 持续时长）→ 氧化速度随温度指数级上升。
  3. **失效形态 + 样品检测**（变脆 + 变色 / 红外光谱 / DSC）→ 区分氧化（变色）vs 水解（保留形态）。
- **典型对话**：
  - 业务员：「客户 PPS 滤袋变脆了，要求索赔。」
  - ✗ 错误回应：「建议换 PTFE 抗氧化。」
  - ✓ 正确回应：「变脆可能是氧化或水解。请先发 ① 袋室前含氧量 ② 实际温度曲线 ③ 失效滤袋样品（送检红外/DSC）。含氧 + 温度 + 形态三项凑齐，才能定是抗氧化还是抗水解。」

## 酸碱腐蚀风险

- 触发信号：存在 SOx、NOx、HCl、碱性气体；滤袋腐蚀、变硬、脆化。
- 需要补充：烟气成分、含湿、温度、露点。
- 不能承诺：不能在烟气成分未知时承诺耐腐蚀方案。
- 建议证据：烟气检测报告、样品检测。
  - **如何拿到**：
    - 烟气检测报告 → 第三方检测（CEMS 在线数据 OR 现场取样化验，标注 SO2/HCl/HF/NOx 浓度 mg/Nm³）
    - 样品检测 → 失效滤袋送检 + 烟气含湿量实测（g/Nm³ 或 %）

- **机制（为什么这个承诺会失败）**：酸碱腐蚀是**特定化学物质 + 含湿/结露**的化学反应——SO2 + H2O → H2SO3 → 腐蚀 PPS、芳纶；HCl 干态对 PPS 几乎无影响但**遇水形成盐酸后腐蚀剧烈**。**关键：腐蚀往往不是滤料本身不耐酸碱，而是"含湿让腐蚀从无到有"**——推荐"耐酸碱滤料"时若不同时点出含湿问题，客户工况下含湿波动时腐蚀再现，承诺被打脸。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **烟气成分定量**（SO2/SO3/HCl/HF/NOx 浓度，mg/Nm³）→ 决定是否真有腐蚀风险、风险等级。
  2. **含湿量 + 露点**（任一缺失即止步）→ 干态 vs 湿态腐蚀速率差 10–100 倍。
  3. **温度区间**（尤其是否有结露风险）→ 酸露点 = 烟气酸腐蚀的真正温度边界。
- **典型对话**：
  - 业务员：「客户烟气有 SO2，要耐酸滤料。」
  - ✗ 错误回应：「建议 PPS 改性耐酸款。」
  - ✓ 正确回应：「SO2 是否形成腐蚀取决于含湿。请先发 ① 烟气成分定量（SO2/HCl/HF 浓度） ② 含湿量 + 露点 ③ 入口温度是否在酸露点以下。成分 + 含湿 + 温度三项凑齐，才能定真正耐腐蚀方案。」

## 粉尘磨蚀风险

- 触发信号：局部破袋、迎风面破损、粉尘硬、流速高。
- 需要补充：粉尘类型、粒径、磨蚀性、过滤风速、破损位置。
- 不能承诺：不能只换材质而忽略气流和结构磨损。
- 建议证据：破损位置照片、粉尘样品、风速数据。
  - **如何拿到**：
    - 破损位置照片 → 现场手机拍摄（迎风面/袋口/袋底/缝合处，标注位置 + 距袋口距离）
    - 粉尘样品 → 客户处取 ≥ 50g 装入密封袋（标注取样位置 + 日期）
    - 风速数据 → 实测过滤风速（m/min），不是设计值——可用风速仪现场测或反算（处理风量 / 总过滤面积）

- **机制（为什么这个承诺会失败）**：磨蚀是**粉尘硬度 + 过滤风速 + 迎风角度**的物理冲刷结果，集中在迎风面（距袋口 0.5–1.5m 段最严重）。**滤料本身的耐磨性差异（PTFE > PPS > 芳纶）远小于风速的差异**——风速从 1.0 m/min 提到 1.5 m/min，磨蚀速率上升 5–10 倍。**只换更耐磨滤料、不降风速、不改气流分布，迎风面还会再磨穿**。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **破损位置分布**（迎风面/袋口/袋底/缝合处）→ 迎风面为主 = 典型磨蚀。
  2. **过滤风速实测**（m/min，不是设计值）→ > 1.2 m/min 进入高磨蚀风险区。
  3. **粉尘硬度 / 粒径**（莫氏硬度 / d50）→ > 莫氏 5（如 SiO2 粉尘）即高磨蚀性。
- **典型对话**：
  - 业务员：「客户迎风面磨穿了，要求换更耐磨滤料。」
  - ✗ 错误回应：「建议 PTFE 基布 + 表面硬化处理。」
  - ✓ 正确回应：「磨穿是风速 + 粉尘 + 迎风角度的综合结果，滤料只是变量之一。请先发 ① 破损位置照片 ② 实测过滤风速（m/min） ③ 粉尘粒径 / 硬度。位置 + 风速 + 粉尘三项凑齐，再决定是降风速、改气流分布还是升级滤料。」

## 粘性 / 油性糊袋风险

- 触发信号：压差上升、清灰无效、粉尘粘或含油。
- 需要补充：粉尘粘性、油性来源、含湿、清灰参数。
- 不能承诺：不能承诺单靠后处理解决全部糊袋。
- 建议证据：糊袋照片、粉尘样品、压差曲线。
  - **如何拿到**：
    - 糊袋照片 → 现场手机拍摄（糊袋区域特写 + 整袋全景）
    - 粉尘样品 → 客户处取 ≥ 50g 装入密封袋（标注取样位置 + 日期，特别标注是否含油/含糖/含粘性杂质）
    - 压差曲线 → DCS/PLC 历史导出（连续 ≥ 7 天，标注清灰时刻）

- **机制（为什么这个承诺会失败）**：粘性/油性糊袋的根因在**粉尘本身的物性**（含油、含糖、含粘性杂质），不在滤料。拒油/拒水后处理只是**降低粘附力**，**不解决粘性粉尘**——粉尘照样累积、照样糊。糊袋的真正解决路径在**上游除油/除尘预处理**（旋风 + 喷淋 + 折流板），不是滤料。**只换"拒油后处理"滤料 = 让客户反复清洗、反复糊袋**。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **粉尘粘性来源**（含油比例 / 糖分 / 树脂类杂质 / 含湿）→ 决定预处理方向。
  2. **上游预处理现状**（旋风、喷淋、折流板是否到位）→ 缺位 = 滤料选型无法解决。
  3. **清灰参数 + 压差曲线** → 区分「清灰无效」vs「粉尘物性过粘」。
- **典型对话**：
  - 业务员：「客户粉尘含油，糊袋严重，要拒油后处理。」
  - ✗ 错误回应：「建议 PTFE 覆膜 + 拒油后处理。」
  - ✓ 正确回应：「拒油后处理只是降低粘附力，不解决粘性粉尘。请先确认 ① 粉尘含油来源（工艺段 + 含油比例） ② 上游是否有旋风/喷淋/折流板预处理 ③ 清灰压力/周期 + 压差曲线。粉尘物性 + 预处理 + 清灰三项凑齐，才能定方向。」

## 火星烧袋风险

- 触发信号：袋面孔洞、局部烧穿、工艺有火星或高温颗粒。
- 需要补充：火星来源、预除尘或阻火措施、峰值温度。
- 不能承诺：不能只靠滤料解决火星进入问题。
- 建议证据：烧损照片、火星捕集记录。
  - **如何拿到**：
    - 烧损照片 → 现场手机拍摄（孔洞/烧穿/局部焦化，附拍摄日期 + 位置标注）
    - 火星捕集记录 → 上游火星捕集器/旋风除尘/喷淋塔的运行记录 + 最近一次清灰/维护日期

- **机制（为什么这个承诺会失败）**：火星烧袋的本质是**火星/高温颗粒进入袋室**——任何滤料（包括 PTFE）在火星持续冲击下都会烧穿。**滤料能挡的是粉尘，挡不了火星**。火星源头不在袋室，滤料管不到；不解决源头（上游火星捕集器、阻火网、喷淋塔），换滤料只是把"短期烧穿"变成"短期再烧穿"。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **火星来源定位**（上游是否有火花捕集器/旋风除尘/喷淋塔，捕集器是否定期清灰）→ 火星源头在袋室之外。
  2. **峰值温度实测**（最近 30 天最高值 + 持续时长）→ 滤料选型的硬边界是峰值温度。
  3. **阻火措施现状**（火花捕集器、阻火网、喷淋塔；最近维护日期）→ 缺位或失效 = 烧袋高风险。
- **典型对话**：
  - 业务员：「客户烧袋了，要求换耐高温滤料。」
  - ✗ 错误回应：「建议 PPS 或 PTFE 滤料。」
  - ✓ 正确回应：「烧袋是火星/超温进入袋室的结果，滤料能挡的有限。请先确认 ① 上游是否有火星捕集措施 + 最近维护日期 ② 最近 30 天峰值温度 + 持续时长 ③ 烧损位置照片。火星源头 + 峰值温度 + 阻火措施三项凑齐，再决定是否需要换滤料。」

## 清灰 / 压差风险

- 触发信号：压差高、清灰频繁、清灰压力异常、喷吹系统异常。
- 需要补充：清灰压力、周期、压差曲线、脉冲阀状态。
- 不能承诺：不能把压差高直接归因于滤料。
- 建议证据：设备运行记录、检修记录。
  - **如何拿到**：
    - 设备运行记录 → DCS/PLC 历史导出（压差/清灰压力/喷吹周期，含时间戳）
    - 检修记录 → 现场检修台账（脉冲阀膜片更换日期、喷吹管检查/疏通日期）

- **机制（为什么这个承诺会失败）**：压差是**清灰效果 + 粉尘层 + 系统阻力**的综合读数。**清灰系统异常（脉冲阀膜片破损、喷吹压力不足、喷吹管堵塞）会让压差虚高**——这种虚高与滤料完全无关。换滤料不修脉冲阀，压差依然虚高。**只看到"压差高"就推"升级滤料"，会延误脉冲阀/喷吹管的检修窗口**——客户工况继续恶化，下一批滤袋也会被错误工况损伤。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **压差曲线**（连续 ≥ 7 天，标注清灰时刻）→ 区分"清灰无效"vs"累积型"。
  2. **清灰系统状态**（喷吹压力、脉冲阀最近检修时间、喷吹管是否堵塞）→ 机械问题先于滤料问题。
  3. **过滤风速实测 + 粉尘浓度** → 排除风速偏高 / 粉尘负荷超设计。
- **典型对话**：
  - 业务员：「客户压差高，要求升级滤料。」
  - ✗ 错误回应：「建议 PTFE 覆膜降低阻力。」
  - ✓ 正确回应：「压差高 70% 是清灰系统问题，不是滤料问题。请先发 ① 压差曲线（标注清灰时刻） ② 清灰压力/周期/脉冲阀状态 ③ 过滤风速实测。机械问题先于滤料问题——脉冲阀/喷吹管先查。」

## 替换降本风险

- 触发信号：客户要求更便宜、同寿命、全量替代、无试用。
- 需要补充：当前方案稳定性、寿命、价格、替换目标、试用接受度。
- 不能承诺：不能承诺低价替代同等寿命。
- 建议证据：竞品样品、历史运行数据、小批量试用结果。
  - **如何拿到**：
    - 竞品样品 → 客户或业务员取竞品滤袋 ≥ 1 条（标注品牌/型号/批号 + 取样日期）
    - 历史运行数据 → 同工况下竞品/我方滤袋的逐月运行记录（实际寿命，不是合同承诺）
    - 小批量试用结果 → 试用报告（含试用时长、对比基线、判定标准、回退条件）

- **机制（为什么这个承诺会失败）**：降本承诺有**两层独立风险**——① 低价方案在客户工况下实际寿命不达标（事后索赔 / 信誉损失） ② "全量替换 + 无试用"会让客户承担一次性失败风险（一旦不达预期，无回退路径）。**承诺"低价 + 同等寿命"在数据上无法支撑**——同寿命 = 同成本（材料 + 工艺 + 后处理），低价 + 同寿命 = 商业上自相矛盾。
- **可执行对策（开口前必查 3 项中的至少 2 项）**：
  1. **当前方案稳定性**（已运行多久、寿命实测、客户满意度）→ 决定替换风险等级。
  2. **替换预算 + 客户对试用的接受度**（是否接受分仓 POC / 小批量试用）→ 决定推进路径。
  3. **竞品 / 替代品实测数据**（同工况历史寿命、材质、工艺）→ 决定"低价同寿命"是否可能。
- **典型对话**：
  - 业务员：「客户要便宜 30%，寿命差不多就行，要求全量替换。」
  - ✗ 错误回应：「可以推荐低成本 PPS 替代。」
  - ✓ 正确回应：「「便宜 30% + 寿命差不多」在数据上无法承诺。请先确认 ① 当前方案已稳定运行多久 + 实测寿命 ② 客户是否接受分仓 POC / 小批量试用 ③ 是否有竞品同工况历史寿命数据。试用边界先定，再谈替换。」

## 跨风险通用检查清单

> 任何风险标签触发时，先过这 3 条再开口。

1. **必查项是否凑齐**？每条风险至少 2 项必查，未凑齐前不归因、不推荐。
2. **是否 framing trap**？客户/业务员已锁定"就是滤料问题"且拒绝看其他可能 → 列入 framing trap，不接受预判。
3. **是否触达合规边界**？涉及寿命 / 排放 / 替换 / 索赔 / 合同 → 走 13-compliance-and-commitment.md，不在本文件给承诺。
