# 02｜工况档案 Schema

工况档案由 Agent 从聊天记录、邮件、会议纪要中抽取，不要求业务员手填。

## 1. 客户与项目背景

| 字段 | 含义 | 业务员可能怎么说 | 缺失影响 |
| --- | --- | --- | --- |
| 客户名称 | 客户或项目名称 | 这个客户、某水泥厂、老客户 | 不影响初判，但影响归档 |
| 行业 | 水泥、钢铁、垃圾焚烧、化工等 | 水泥厂、钢厂、焚烧炉 | 影响常见风险路径 |
| 工艺段 | 窑尾、窑头、烧结、锅炉、破碎等 | 窑尾袋子、烧结机后面 | 影响温度、粉尘、腐蚀判断 |
| 项目类型 | 新项目、故障、替换、无法分流 | 新上项目、现场坏了、想换便宜 | 决定追问路径 |
| 业务员原始描述 | 业务员粘贴或转述内容 | 客户说袋子不耐用 | 保留输入语境 |
| 客户原话 | 客户自己的表达 | 原文聊天记录 | 缺失会增加误判风险 |
| 本次目标 | 这轮要解决什么 | 报价、判断原因、给客户回复 | 决定输出深度 |

## 2. 当前运行状态

| 字段 | 含义 | 缺失影响 |
| --- | --- | --- |
| 是否已有除尘系统 | 新项目或已运行项目 | 决定是选型还是故障诊断 |
| 除尘器类型 | 袋式除尘器类型或设备结构 | 影响滤袋尺寸、清灰和运行判断 |
| 清灰方式 | 脉冲、反吹、机械振打等 | 影响压差、糊袋、破袋判断 |
| 风量 | 系统处理风量 | 影响过滤风速判断 |
| 过滤风速 | 气布比 | 影响寿命、压差、排放 |
| 入口 / 出口压差 | 运行阻力 | 影响糊袋、清灰、堵塞判断 |
| 运行时间 | 已运行时长 | 影响寿命判断 |
| 启停频率 | 是否频繁启停 | 影响结露和水解风险 |
| 近期工况变化 | 原料、燃料、工艺变化 | 影响故障归因 |

## 3. 烟气与粉尘条件

| 字段 | 含义 | 缺失影响 |
| --- | --- | --- |
| 常态温度 | 长期运行温度 | 影响材质耐温方向 |
| 峰值温度 | 短时最高温 | 影响烧袋和热收缩风险 |
| 含湿量 | 烟气水分 | 影响结露、水解、糊袋 |
| 是否结露 | 是否低于露点或启停结露 | 影响糊袋和腐蚀判断 |
| 酸碱 / 腐蚀气体 | SOx、NOx、HCl、碱性气体等 | 影响腐蚀和材质边界 |
| 含氧量 | 氧化环境强弱 | 影响氧化风险 |
| 粉尘类型 | 水泥、生物质灰、炭黑、金属粉等 | 影响磨蚀、粘性和排放 |
| 粉尘细度 | 粒径或细粉比例 | 影响过滤和排放 |
| 磨蚀性 | 粉尘是否冲刷强 | 影响破袋和寿命 |
| 粘性 / 油性 | 是否易粘附 | 影响糊袋和清灰 |
| 火星 / 高温颗粒 | 是否有火星进入 | 影响烧袋 |
| 易燃易爆风险 | 粉尘爆炸或可燃风险 | 影响安全边界 |

## 4. 当前滤袋 / 滤料信息

| 字段 | 含义 | 缺失影响 |
| --- | --- | --- |
| 当前材质 | PPS、PTFE、P84、芳纶、涤纶、玻纤等 | 影响故障和替换判断 |
| 克重 | 单位面积重量 | 影响强度和透气 |
| 后处理 | 覆膜、防水防油、烧毛、热定型等 | 影响糊袋、排放和清灰 |
| 规格尺寸 | 滤袋尺寸 | 影响报价和替换 |
| 供应商 / 竞品 | 当前供应来源 | 影响竞品替换策略 |
| 已使用时长 | 当前寿命 | 影响是否异常 |
| 历史寿命 | 过往平均寿命 | 影响改善目标 |
| 失效形态 | 糊、破、烧、硬化、腐蚀 | 影响根因判断 |
| 更换频率 | 多久换一次 | 影响成本和寿命目标 |
| 样品 / 照片 / 检测报告 | 证据材料 | 缺失时不能下定论 |

## 5. 客户目标与约束

| 字段 | 含义 | 缺失影响 |
| --- | --- | --- |
| 客户想解决的问题 | 客户主观问题 | 用于理解客户诉求 |
| 客户表达的需求 | 客户要什么 | 可能不是真实问题 |
| 真实工况问题假设 | Agent 初步诊断 | 需要证据支持 |
| 目标寿命 | 客户期望寿命 | 影响方案方向 |
| 排放要求 | mg/Nm3 或地方标准 | 影响覆膜和过滤精度 |
| 成本目标 | 降本或预算边界 | 影响替换策略 |
| 交期要求 | 项目时间 | 影响可执行性 |
| 验收标准 | 如何判断成功 | 没有则不能做确定承诺 |
| 预算 / 采购阶段 | 是否进入采购 | 影响下一步推进 |

## 6. 已知 / 未知 / 矛盾信息

Use these labels in output:

- `[已知]`
- `[未知]`
- `[矛盾]`
- `[需客户确认]`
- `[需证据]`

## 7. 初步风险标签

Use risk tags from `04-risk-tags.md`.

## 8. 判断等级

Always output:

- 当前 Level。
- 当前允许输出。
- 当前禁止输出。
- 下一步动作。
